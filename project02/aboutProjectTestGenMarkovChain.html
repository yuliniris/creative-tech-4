Responses to brief:

What kind of work will you have your bot do for you

I set four kinds of tasks for chatbot:

  • Response by showing bottons
  • Response by sending text message
  • Response by showing dialogue box and calling functions according to answers(parameters)
  • Response by fetching open API

    • How is your bot triggered? How will people interact with your bot?

    My bot triggered by sending text message 'would you like to have a chat?'.
    Users can interact with it by two ways:

  • Clicking bottons: 'Any questions', 'Continue', 'Have a break'
  • When clicking 'Continue', it will show the dialogue box, and users need to enter answers as the parameters to call the Text Generation function.

    • What kind of personality will you give this bot? How will it have conversations with the person interacting with it?

    I use the scripts of Game of Thrones to train the Markov Chain model. i hope it can tell more about this story.
    Users need to start the conversation by sending text messages, and anything is okay. Chatbot may pick up the last three letters as the start to generate new vocabulary.
    Also, users need to tell chatbot the length of paragraph.
    Additionally, the caculation units of three letters can be changed as well. if setting the units length as four, the model will caculate the probility of the fifth character. Since, in the train part, this parameter is set three, right now the unit is three letters.

    What kind of algorithms will you use to give life to your bot?

    I use Markov Process to generate text, and i apply two methods.
    First one is using the existing library called Markovify. Second one is writing the Markov Process step-by-step. The model caculates the probilities table and then using the probilities table to generate new texts.
